LUAR- Tienda de medias

DESCRIPCIÓN
LUAR es un sitio web desarrollado como trabajo práctico de la materia de Programación III. El objetivo del proyecto es ofrecer una tienda online dedicada a la venta de medias deportivas, urbanas, brindando una navegación simple, un diseño moderno y una experencia intuitiva para el usuario.

FUNCIONALIDADES.
-Página de inicio con presentación de la marca.
-Catálogo de productos.
-Formulario de contacto con validaciones en JavaScript.
-Página de inicio de sesión.
-Diseño adaptable a distintos dispositivos.
-Navegación entre todas las secciones del sitio.

TECNOLOGÍAS UTILIZADAS
- HTML5
- CSS3
- JavaScript
- Bootstrap 5

CÓMO EJECUTAR EL PROYECTO
1. Clonar o descargar este repositorio.
2. Abrir la carpeta del proyecto en Visual Studio Code (opcional).
3. Ejecutar el archivo index.html desde un navegador web.
4. También se puede acceder al sitio desde GitHub Pages una vez publicado.

ESTRUCTURA DEL PROYECTO
° index.html: Página principal.
° productos.html: Catálogo de productos.
° contacto.html: Formulario de contacto.
° login.html: Inicio de sesión.
° css/: Archivos de estilos.
° js/: Scripts y validaciones.
° Imagen: Imágenes utilizadas en el sitio.
